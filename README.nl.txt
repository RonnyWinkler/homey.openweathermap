Deze app brengt de OpenWeatherMap API naar Homey.
Ontvang actuele weergegevens en voorspellingen voor een willekeurig aantal locaties.

De volgende weergegevens zijn beschikbaar:
- Actuele weergegevens
- Uurvoorspellingen tot 48 uur
- Dagelijkse voorspellingen tot 7 dagen

Voor elke variant zijn verschillende waarden voorzien, zoals temperatuur, luchtdruk, vochtigheid, weersomstandigheden en vele andere waarden.

Een beschrijving van de API en voorbeeldgegevens zijn te vinden op https://openweathermap.org/api/one-call-api

Voorwaarde:
Een API-sleutel is vereist om de OpenWeatherMap API te gebruiken.
De API-sleutel kan gratis worden gedownload van https://openweathermap.org/api.
Per API-sleutel zijn 1000 calls per dag toegestaan.

Conversie naar de OneCall API:
De vorige Homey-apparaten gebruiken een oudere API-versie die in de toekomst niet meer door OWM wordt ondersteund of alleen beschikbaar is in abonnementen.
Deze apparaten zijn gemarkeerd als [verouderd]. Ze werken nog steeds, maar ze kunnen niet meer worden toegevoegd. 
Zolang de API-aanroepen nog mogelijk zijn, kunnen deze toestellen gebruikt blijven worden.
Het wordt echter aanbevolen om deze apparaten te vervangen door de nieuwe apparaten en de stromen aan te passen.
